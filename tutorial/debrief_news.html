It's easy to keep up to date with what's happening in Debrief. You can find out about new bugs, software changes, and formal software releases using one of these methods:

14.2.1. Twitter

Twitter fans just have to follow the @debrieftool account.

14.2.2. Email

Alternatively, you can navigate to Debrief's GitHub site (https://github.com/debrief/debrief ), and then either Star the page to hear high level announcements, or Watch the page to hear all the gory detail. Learn more about the options here.
