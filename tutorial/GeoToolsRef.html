17.1. Introduction

The display of chart data in Debrief is all down to the GeoTools library. Recently the SWT module was added to this library. The SWT module incorporates the SwtMapPane class upon which the Debrief plot now sits. With the incorporation of GeoTools brings a wide range of new file formats, both image and vector, plus more advanced topographic and earth model related capabilities. Rest assured, Debrief will exploit these library features as much as possible in coming months/years.
