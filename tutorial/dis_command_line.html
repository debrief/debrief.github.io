20.2. Command line options

Some organisations may wish to specify the DIS IP Address & Port values in the command line when starting Debrief. This allows a parent process/simulator to start Debrief preconfigured with the correcft network parameters.

Set these values by using these settings on the command line: -Dport=2345 -DipAddress=23.44.55.66.
